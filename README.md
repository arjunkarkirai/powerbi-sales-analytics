# powerbi-sales-analytics
Power BI • SQL • DAX • Data Modeling • Business Insights  This repository contains an end-to-end Data Analytics project built using SQL and Power BI, designed to replicate real-world business analytics in retail/e-commerce environments.
Project Objectives

Build a complete analytics pipeline from raw data → insights.

Design a star schema for efficient BI reporting.

Develop key business KPIs using DAX.

Build interactive dashboards for sales, product, customer, returns, and campaigns.

Present actionable business insights for decision-makers.

**Business Insights Delivered
**
The top 10 products contribute over 60% of revenue → stock priority recommended.

Low stock alerts help avoid lost sales due to out-of-stock situations.

Product category “X” has a significantly higher return rate, indicating defect or expectation mismatch.

Campaign ROI varies heavily — Campaign B underperforms and requires budget review.

The top 20% of customers contribute most sales → key group for loyalty & retention efforts.
